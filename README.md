# 🍲 NGO Document Automator

A specialized tool designed to assist an NGO providing food to approximately **150 people** in need each month. This script bridges the gap between traditional paperwork and modern efficiency without disrupting established workflows.

## 📋 The Problem
The NGO currently handles a high volume of paperwork for public institution compliance:
*   **Manual Effort:** Creating recipient forms manually and filling in product columns by hand.
*   **Repetitive Tasks:** Managing 2-3 documents per person, often reusing forms for up to 6 months.
*   **Time Sink:** Paperwork management takes roughly **15–20 hours** per cycle.
*   **Tech Barrier:** Moving to a fully digital system can be overwhelming for staff used to existing PDF and Excel workflows.

## 🚀 The Solution
This HTML-based script automates document generation while remaining **highly adapted** to the NGO's current processes. It allows the team to save hours of manual labor without having to learn complex new software.

### Key Benefits
*   **Zero Learning Curve:** Staff can continue using their existing PDF templates and Excel lists.
*   **Massive Time Savings:** Reduces paperwork from 20 hours to just a few minutes.
*   **Accuracy:** Eliminates manual entry errors in product lists and personalized info.

---

## 🛠 How It Works
The script follows a simple, three-step logic:

1.  **Template Upload:** Provide your existing PDF form template.
2.  **Recipient Data:** Copy/paste the list of people (including personalized info).
3.  **Product Selection:** Drag and drop the Excel file containing the products for the current serving.

### What stays the same?
The NGO can continue to reuse:
1.  📄 **PDF Form Templates**
2.  👥 **Existing Recipient Lists**
3.  📊 **Monthly Excel Product Lists**

---

## ✅ Result
The script generates a **personalized PDF document** for every individual on the list. Each document is automatically populated with the specific products corresponding to that person's status.

---

### 💻 Technologies Used
*   **HTML/JavaScript:** For the automation logic and user interface.
*   **PDF Integration:** For handling and filling existing templates.
*   **Excel Parsing:** For reading monthly product data.
